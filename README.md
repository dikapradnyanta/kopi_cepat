# KopiCepat

Proyek UTS — simulasi aplikasi pemesanan kopi online. Frontend only, tidak ada backend.

Dibuat pakai Next.js 14, TypeScript, dan Tailwind CSS.

---

## Yang ada di dalamnya

- Browse menu kopi + filter kategori
- Kustomisasi pesanan (gula, es, ukuran)
- Keranjang + simulasi checkout & pembayaran
- Rating dan ulasan per produk  
- Rekomendasi menu acak
- Support dark mode

## Jalankan lokal

```bash
npm install
npm run dev
```

Akses di `localhost:3000`
